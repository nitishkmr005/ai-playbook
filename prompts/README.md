# Prompt Collection

Organized prompts from various sources — images, links, and text snippets.

## Structure

| Folder | Contents |
|--------|----------|
| `image-generation/` | Prompts for generating images: infographics, 3D illustrations, logos, artistic styles, character design |
| `ui-design/` | Prompts for designing UI/UX mockups and SaaS interfaces |
| `meta-prompts/` | Prompts about prompts — evaluators, builders, memory systems |
| `reasoning/` | Structured reasoning frameworks (tree-of-thought, etc.) |
| `content-creation/` | Prompts for scripts, articles, and media |
| `references/` | External resources and system prompt repositories |
