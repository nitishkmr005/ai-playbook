## Comic-Style Infographic

> Create a comic-style infographic summarizing the given text.
>
> Use a clean, modern layout with a soft cream background, navy blue text, and yellow-highlighted section headers and insight boxes.
>
> Structure the image with the following components:
>
> 1. **Title:** Display the main title in bold, large font at the top.
> 2. **Section Blocks:** For each key concept or category in the text, create a rounded yellow-highlighted header (e.g., "Method", "Focus Tips", "Insights", etc.). Under each, use 3–5 bullet points for clarity.
> 3. **Icons:** Add simple flat-style icons next to each section (e.g., notebook, target, calendar, brain, writing hand, etc.)
> 4. **Key Insight Box:** Include a rounded yellow box at the bottom with a casual quote or a paraphrased insight in a human, reflective tone.
>
> Design should be visually appealing for educational or productivity-themed social media posts. Use soft shadows, rounded shapes, minimal clutter, and balanced whitespace.

**Tags:** `comic`, `educational`, `social-media`, `summarization`

## Postcall Rep Note Generation — Process Infographic

> Create a clean, professional horizontal 3D infographic titled 'Postcall Rep Note Generation' on a soft gradient background (ivory → light blue). Use smooth clay-style textures, gentle drop shadows, and modern sans-serif labels. Arrange six equally spaced blocks in a strict left-to-right order:
>
> 1. Call Transcript – a paper icon overlaid with an audio-waveform graphic
> 2. LLM Model – a stylized neural-chip icon
> 3. Information Extraction – a form or data-table icon
> 4. Intermediate Rep Note – a green spiral-notebook icon
> 5. Claude LLM – the official Claude logo/icon
> 6. Final Rep Note – a friendly 3D character at a desk holding a concise note
>
> Connect each block with a single curved arrow, alternating blue (➔) and orange (➔):
>
> - From Call Transcript to LLM Model, a blue arrow labeled "Prompt 1" directly above the arrow.
> - From LLM Model to Information Extraction, an orange arrow (no label).
> - From Information Extraction to Intermediate Rep Note, a blue arrow labeled "Extract Key Fields" above it.
> - From Intermediate Rep Note to Claude LLM, an orange arrow labeled "Prompt 2" above it.
> - From Claude LLM to Final Rep Note, a blue arrow (no label).
>
> Ensure arrows point precisely to the center of each target block, labels are legible, and the overall flow visually reads left to right without any loops. Use a consistent pastel palette and maintain spacing so the sequence is crystal-clear to business stakeholders.
