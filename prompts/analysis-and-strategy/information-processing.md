# Information Processing — Beyond "Summarize"

Stop asking AI to summarize. These 8 prompts extract more value from the same information.

Source: Whiteboard infographic

---

## 1. Extract Strategic Insights

*Instead of:* "Summarize this."

```
Act like a strategy consultant. Identify the 5 most valuable insights and explain what decisions each one informs.
```

**Use when:** You have data, a report, or a long document and need to know what to *do*, not just what it says.

---

## 2. Turn Information Into Action

*Instead of:* "Give me the key points."

```
Translate this into a 5-step action plan with clear owners, quick wins, and measurable results.
```

**Use when:** You need to move from reading to doing. Converts passive content into an execution checklist.

---

## 3. Surface Hidden Assumptions

*Instead of:* "What does this say?"

```
Reveal the unstated assumptions or blind spots shaping this argument — and what changes if they're wrong.
```

**Use when:** Evaluating a business case, research paper, proposal, or argument where you need to stress-test the logic.

---

## 4. Compare Opposing Views

*Instead of:* "Explain this topic."

```
Map this idea against two competing perspectives. Show where they align, where they differ, and which context fits each.
```

**Use when:** Making a decision between two approaches, evaluating a debate, or understanding a contested topic.

---

## 5. Distil for a Specific Role

*Instead of:* "What's important here?"

```
Filter this through the lens of a [ROLE]. For example:
- Marketer → Reach/Conversion implications
- Founder → Cash flow/Growth implications
- Engineer → Feasibility/Scale implications
```

**Use when:** The same information means different things to different stakeholders. Tailor the insight to the audience.

**Variables:** `[ROLE]`

---

## 6. Build a Reusable Model

*Instead of:* "Can you explain the process?"

```
Extract the repeatable framework hidden in this text. Label each stage, its input, and its output.
```

**Use when:** You want to apply the same logic to future situations — turn a one-off insight into a system.

---

## 7. Extract Contrarian Takeaways

*Instead of:* "What are the main lessons?"

```
Find insights that would challenge smart peers — still credible, but unexpected. Write each as a sharp one-liner.
```

**Use when:** Preparing to present, write, or persuade. Contrarian insights are memorable and signal deep thinking.

---

## 8. Identify Leverage Points

*Instead of:* "What should I focus on?"

```
Highlight the 3 leverage points where small actions could create outsized results. Explain why each one matters.
```

**Use when:** You have limited time/resources and need to find the highest-impact interventions.
